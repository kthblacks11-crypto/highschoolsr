💡 API 키란 무엇인가요?

이 시스템은 구글의 최신 인공지능을 사용해 문제를 정밀하게 분석합니다. 인공지능을 무료로 사용하기 위한 일종의 '개인 전용 출입증'이 바로 API 키입니다.

🚀 1분 만에 API 키 발급받는 방법 (초보자용)

⚠️ 교육청/학교 공공 계정 사용자 필독!

교육청 계정(예: @cne.go.kr 등)은 보안 정책상 API 발급 사이트 접속이 차단되어 있을 수 있습니다.
접속 시 '권한 없음' 오류가 뜬다면, 당황하지 마시고 개인 스마트폰 구글 계정(@gmail.com)으로 로그인하여 발급받은 뒤 키를 복사해 주세요. (개인 계정의 키를 사용해도 시스템은 완벽하게 작동합니다!)

  1. 아래 파란색 링크를 클릭해 구글 AI 스튜디오로 이동하세요. (구글 로그인 필요)
    👉 [구글 AI 스튜디오 바로가기]
  2. 화면 오른쪽 위에 있는 [API 키 만들기] 버튼을 클릭하세요.
  3. (핵심) 화면 가운데 '새 키 만들기' 창이 뜹니다. 'Default Gemini Project' 을 그대로 두고 바로 아래의[키 만들기]버튼을 누르세요.
    * 만약 여러 곳에 API를 사용하시는 경우 [+ 프로젝트 만들기]를 선택해서 API키를 구분해서 관리하세요.(키 생성까지 5~10초 소요)
  4. 'AIzaSy...' 로 시작하는 아주 긴 비밀번호가 생성됩니다. 그 옆의 [복사 아이콘(문서가 겹친 모양)]을 누르세요.
  5. 다시 이 화면으로 돌아와서, 아래 빈칸에 복사한 키를 붙여넣기(Ctrl+V) 하세요!
