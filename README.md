# 2022 개정 수학 성취기준 매칭 시스템 🔍

이 시스템은 수학 문제 이미지를 AI(Gemini)로 분석하여 **2022 개정 교육과정 성취기준**과 매칭하고, 저작권을 준수하기 위해 문제를 변형해주는 도구입니다.

## 🚀 시작하기 (이용자 가이드)

### 1. AI 분석 설정 (무료)
본 서비스는 이용자의 개인 API 키를 사용하여 비용 없이(무료 티어) 분석을 수행합니다.
1. [Google AI Studio](https://aistudio.google.com/app/apikey)에 접속합니다.
2. **'Create API key'** 버튼을 클릭하여 키를 발급받습니다.
3. 서비스 우측 상단의 **설정(⚙️) 아이콘**을 눌러 키를 붙여넣으세요.
   * *주의: 입력하신 키는 브라우저에 안전하게 암호화되어 저장되며 서버로 전송되지 않습니다.*

### 2. 내 문제 분석하기
* **업로드**: 문제 사진을 찍어 올리거나, `Ctrl+V`로 복사-붙여넣기 하세요.
* **분석 결과**: AI가 문제의 핵심 개념을 파악하고, 저작권 보호를 위해 숫자와 상황을 바꾼 **새로운 유사 문제**를 만들어줍니다.
* **학습 활용**: 변형된 문제는 데이터베이스에 저장되어 다른 이용자들의 '문항 매칭 연습' 자료로 활용됩니다.

## 🛠️ 자주 묻는 질문 (FAQ)

**Q. 분석 중 'RESOURCE_EXHAUSTED' 오류가 발생해요.**
> 구글 AI Studio의 무료 할당량(분당 일정 횟수)을 초과한 경우입니다. 1분 정도 기다렸다가 다시 시도하거나, 다른 구글 계정으로 새 키를 발급받으세요.

**Q. 사진 인식이 잘 안 돼요.**
> 수식이 포함된 부분은 밝고 선명하게 찍어주세요. 그림이나 그래프가 복잡할 경우 텍스트 위주로 캡처하면 인식률이 올라갑니다.

**Q. API 요청 말고 다른 방법은 없나요?**
> 현재로서는 가장 정확하고 비용이 들지 않는 방법이 API 방식입니다. 대안으로는 **로컬 OCR(문자 인식)** 기술이 있으나, 수학 문제의 깊이 있는 분석이나 저작권 보호를 위한 문제 변형은 AI API가 가장 뛰어납니다. 향후 서버 비용 부담이 없는 한도 내에서 공용 키 제공을 검토하겠습니다.
