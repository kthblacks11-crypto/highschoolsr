⚙️ 구글 AI API 키 설정

💡 API 키란 무엇인가요?

이 시스템은 구글의 최신 인공지능을 사용해 문제를 정밀하게 분석합니다. 인공지능을 무료로 사용하기 위한 일종의 '개인 전용 출입증'이 바로 API 키입니다.

🚀 1분 만에 API 키 발급받는 방법 (초보자용)

  1. 아래 파란색 링크를 클릭해 구글 AI 스튜디오로 이동하세요. (구글 로그인 필요)
    👉 [구글 AI 스튜디오 바로가기]
  2. 화면 오른쪽 위에 있는 회색 [API 키 만들기] 버튼을 클릭하세요.
  3. (핵심) 화면 가운데 '새 키 만들기' 창이 뜹니다. '가져온 프로젝트 선택' 칸을 누르고 [+ 프로젝트 만들기]를 선택한 뒤, 바로 아래의 [키 만들기] 버튼을 누르세요.
    * 만약 이미 영문으로 된 프로젝트(예: Default Gemini Project)가 있다면 아무거나 선택하셔도 무방합니다. (키 생성까지 5~10초 소요)
  4. 'AIzaSy...' 로 시작하는 아주 긴 비밀번호가 생성됩니다. 그 옆의 [복사 아이콘(문서가 겹친 모양)]을 누르세요.
  5. 다시 이 화면으로 돌아와서, 아래 빈칸에 복사한 키를 붙여넣기(Ctrl+V) 하세요!

⚠️ 사용 중 '무료 할당량 초과' 에러가 뜬다면?

한도를 다 썼다는 뜻이니 당황하지 마세요! 위 사이트에 다시 접속해서 똑같이 새 키를 하나 더 발급받아 아래 칸에 새로 덮어쓰기 해주시면 계속 무료로 사용할 수 있습니다.
